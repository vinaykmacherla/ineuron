This Project has to applications:
app.py - Demo app for how Flask works
app_json.py - Using Flask + json request through postman client

app_json.py :
This demo Flask app is to run the API in postman client through json request/response.
We creaet a json input and ask to provide the json output from server.

Do request from postman in json format as below.

{

"operation":"multiply",
"number1":10,
"number2":20

}
